# balancer_hw

hardware (pcb, 3d designs, ...) for balancer robot

## cad

openSCAD models

+ bal01: oldest version, not in use anymore
+ bal02: working copy of bal01, this will change a lot in future

## kicad

PCBs in KiCad

+ bal01: rotary sensor for motor speed, MA70x, used with G431B-ESC1
+ bal02: mainboard sporting STM32L432KB Nucleo-32
